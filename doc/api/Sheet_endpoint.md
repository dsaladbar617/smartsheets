# Sheet API
## [Change base sheet data](./API/get_all_users.md)
```js
PATCH api/edit_sheet/:sheet_id
```
***
## [Add a new sheet](./API/get_all_users.md)
```js
POST api/add_sheet
```
***
## [Get all sheets current user is part of](./API/get_all_users.md)
```js
GET api/get_sheets
```
***
